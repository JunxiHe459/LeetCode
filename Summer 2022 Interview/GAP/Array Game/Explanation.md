# Array Game

给一个 array 比如 [3, 4, 6, 6, 3]

每次选一个元素不变，其余元素全部 + 1

问：最少多少次操作，可以让所有元素相等

例如 [3, 4, 6, 6, 3]:

[4, 5, 7, 6, 4]

[5, 6, 7, 7, 5]

[6, 7, 8, 7, 6]

[7, 8, 8, 8, 7]

[8, 8, 9, 9, 8]

[9, 9, 10, 9, 9]

[10, 10, 10, 10, 10]

需要 7 次

做题前：

1. 遇到这种算次数题，肯定不要暴力去解，绝对超时

做题中：

1. 假如只有 [3, 4]，需要 1 次

2. 假如只有 [3, 4 ,6] 首先 1 次变为 [4, 4, 7], 然后 3 次变为 [7, 7, 7]

3. 假如只有 [3, 4, 6, 6] 首先 1 次变为 [4, 4, 7, 7], 然后 3 次变为 [7, 7, 7, 10], 然后 3 次变为 [10, 10, 10, 10]

4. 通过以上 3 个例子发现：总共的次数 = Sum (arr[i] - min(arr))

   ```python
   def ArrayGame(arr):
       least = min(arr)
       count = 0
       for i in arr:
           count += i - least
       return count
   ```
